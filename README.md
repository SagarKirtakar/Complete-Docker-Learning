# 🐳 Complete Docker Learning for Java Developers

A beginner-friendly repository to learn **Docker from scratch** with a focus on **Java & Spring Boot** applications.

## 📚 About

This repository contains my Docker learning journey, including theory, hands-on practice, Docker commands, Docker architecture, and containerizing Spring Boot applications.

The goal of this repository is to help Java developers understand Docker concepts through practical examples.

## 🚀 Topics Covered

* Docker Basics
* Docker Architecture
* Docker Engine
* Docker Images & Containers
* Docker Hub
* Dockerfile
* Docker Volumes
* Docker Networks
* Docker Commands
* Spring Boot with Docker
* Running Spring Boot without Dockerfile
* Running Spring Boot using Dockerfile
* Port Mapping
* Container Lifecycle

## 🛠️ Technologies Used

* Java
* Spring Boot
* Maven
* Docker
* OpenJDK
